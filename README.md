# Reh Dogg Ski Rush

A neon arcade-style downhill skiing game created for Reh Dogg Games Plus.

## Features

- Arcade skiing gameplay
- Increasing difficulty system
- Avalanche warning events
- Ice patch hazards
- Shield pickups
- Neon arcade visuals
- Mobile touch controls
- Keyboard support
- WebAudio sound effects
- Future Suno music support
- GitHub Pages compatible

## Controls

### Desktop
- LEFT ARROW / A = Move Left
- RIGHT ARROW / D = Move Right
- ENTER or SPACE = Start Game

### Mobile
- Touch buttons on screen
